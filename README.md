【游戏名称】台球游戏

【游戏介绍】
这是一款H5原生开发的体育类小游戏。游戏采用Canvas 2D渲染引擎开发，提供流畅的台球游戏体验。玩家需要控制台球杆，击打白球，将彩色球击入球袋中。游戏具有真实的物理引擎，模拟真实的台球碰撞和运动轨迹。

【玩法说明】
操作：
- 在移动设备上，使用屏幕上的虚拟控制器
- 点击"⚡"按钮调整击球力度（长按增加力度）
- 点击"🎱"按钮击球
- 点击"🔄"按钮重新开始游戏
- 在桌面端，使用鼠标操作
- 鼠标左键按住调整击球力度，松开左键击球

规则：
- 使用白球击打彩色球
- 将彩色球击入球袋中
- 每次击球后，球会自然停止
- 将所有彩色球击入袋中即可获胜
- 不能将白球击入袋中，否则失败

目标：
- 将所有彩色球击入球袋中
- 获得更高的击球准确率
- 完成游戏后可以重新开始挑战更高分数

Tips：
- 观察球的轨迹和角度，预判击球效果
- 适当控制击球力度，不要过于用力
- 利用台边进行反弹，创造更好的击球角度
- 规划击球顺序，优先击打容易命中的球
- 多次尝试不同角度和力度，找到最佳击球策略

技术介绍：
- 采用Canvas 2D渲染引擎，提供流畅的游戏画面
- 物理引擎模拟真实的台球碰撞和运动轨迹
- 响应式设计，完美适配移动端和桌面端
- 虚拟控制器专为移动设备优化，提供直观的触摸操作
- 高性能渲染系统，支持流畅的动画效果
- 自适应屏幕尺寸，确保在各种设备上的最佳显示效果
